# Icons et assets statiques

Ce dossier contient les assets statiques de l'application.

## À faire

- [ ] Ajouter `favicon.png` (16x16, 32x32, 48x48)
- [ ] Ajouter `icon-192.png` pour PWA
- [ ] Ajouter `icon-512.png` pour PWA
- [ ] Ajouter `apple-touch-icon.png`
- [ ] Ajouter images de splash screen PWA si nécessaire

## Génération des icons

Vous pouvez utiliser des outils comme:

- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)

Pour l'MVP, vous pouvez créer des icons simples avec les initiales "RA" sur fond bleu (#0071e3).
